# Crediti icone

Le icone in questa cartella provengono da [Iconoir](https://iconoir.com),
libreria open source di icone SVG (licenza MIT).

| File              | Sorgente                                |
|-------------------|------------------------------------------|
| `switch-on.svg`   | iconoir.com (variante "Switch On")       |
| `switch-off.svg`  | iconoir.com (variante "Switch Off")      |

Le icone sono usate via CSS mask in `style.css` (vedi `button.switch`),
così il colore arriva dal tema (`background-color`) e segue il colore
semantico del contesto. Vedi `regole di design.md §3` per la variante
`switch` di `button`.
